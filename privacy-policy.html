Privacy Policy for Font Detector Chrome Extension

Data Collection

This extension does NOT collect, store, or transmit:

  • Personal user information
  • Browsing history
  • Website content
  • Any sensitive data

Data Usage

The extension may store locally:

  • User preferences (through Chrome storage)
  • Font statistics (optional opt-in feature)

No data is shared with third parties except for:

  • Font verification through official font service APIs (Google Fonts, Adobe Fonts, Typekit)

Permissions Justification

  • activeTab: Required to analyze fonts on current webpage
  • scripting: Needed for font detection functionality
  • storage: Used for saving user preferences
